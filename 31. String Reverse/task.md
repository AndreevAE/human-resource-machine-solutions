
# String Reverse

## Task

For each zero terminated string in the `INBOX`, reverse it and put the result in the `OUTBOX`. Repeat!

## Example

INBOX:  
`B` `U` `G` `0` `Y` `O` `U` `0` `Y` `O` `U` `0`

OUTBOX:  
`G` `U` `B` `U` `O` `Y` `U` `O` `Y`
