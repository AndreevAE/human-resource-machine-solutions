
# Re-Coordinator

## Task

Each number in the `INBOX` is an address of a tile on the floor. Send to the `OUTBOX` the coordinates of that tile, column first, row second.

For example, an address of 6 has coordinates 2, 1. You may ask your boss for more examples.

## Example

INBOX:  
`8` `11`

OUTBOX:  
`0` `2` `3` `2`

## Optimizaiton Challenges

**Size Challenge:** 14  
Measures the number of commands in your program. Fewer is better.

**Speed Challenge:** 76  
Measures the number of commands your program must execute before it finishes, on average. Fewer is better. (Average computed by testing your solution against many deterministically generated input sets.)
